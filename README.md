## Run Locally

### 1. Clone repo

```
$ git clone https://github.com/chiemelaumeh/chiemela_furniturestore.git


```

### 2. Create .env File

- rename the .env.example file in backend folder to .env

### 3. Setup .env file with application credientials provided in the project document

### 4. Run Backend

```
$ cd server
$ npm install
$ nodemon server.js
```

### 5. Run Frontend

```
# open new terminal
$ cd client
$ npm install
$ npm start
```
